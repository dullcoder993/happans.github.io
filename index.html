Linked List

Definition

A Linked List is a linear data structure where elements (nodes) are connected using pointers. Each node contains data and a reference (link) to the next node.

Algorithms

Insert at Beginning:

  1. Create a new node with the given value
  2. Set the new node's next pointer to the current head
  3. Update the head to point to the new node

Search Operation:

  1. Start from the head node
  2. Traverse through each node until you find the target value
  3. If found, return the node; else return null

Time Complexity

Access: O(n)
Search: O(n)
Insertion: O(1) at head, O(n) at tail
Deletion: O(1) at head, O(n) at tail

Common Operations

Insertion (at beginning, end, or specific position)
Deletion (by value or position)
Searching for an element
Reversing the list

Code (C Example)

#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node* next;
};

void insertFront(struct Node** head, int value) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = value;
    newNode->next = *head;
    *head = newNode;
}

void printList(struct Node* head) {
    while (head != NULL) {
        printf("%d -> ", head->data);
        head = head->next;
    }
    printf("NULL\n");
}

int main() {
    struct Node* head = NULL;
    insertFront(&head, 10);
    insertFront(&head, 20);
    insertFront(&head, 30);
    printList(head);
    return 0;
}

Output

30 -> 20 -> 10 -> NULL

Stack (LIFO)

Definition

A Stack is a linear data structure that follows LIFO (Last In, First Out) principle.

Insertion = Push | Deletion = Pop

Algorithms

Push Operation:

  1. Check if stack is full (overflow condition)
  2. Increment the top pointer
  3. Insert the element at stack[top]

Pop Operation:

  1. Check if stack is empty (underflow condition)
  2. Return the value at stack[top]
  3. Decrement the top pointer

Time Complexity

Access: O(n)
Search: O(n)
Insertion (Push): O(1)
Deletion (Pop): O(1)

Common Operations

Push (add element to top)
Pop (remove element from top)
Peek (view top element)
Check if stack is empty or full

Code (C Example)

#include <stdio.h>
#define MAX 5

int stack[MAX], top = -1;

void push(int value) {
    if (top == MAX-1)
        printf("Stack Overflow\n");
    else
        stack[++top] = value;
}

void pop() {
    if (top == -1)
        printf("Stack Underflow\n");
    else
        printf("Popped: %d\n", stack[top--]);
}

int main() {
    push(10);
    push(20);
    push(30);
    pop();
    return 0;
}

Output

Popped: 30

Queue (FIFO)

Definition

A Queue is a linear data structure that follows FIFO (First In, First Out) principle.

Insertion = Enqueue (rear) | Deletion = Dequeue (front)

Algorithms

Enqueue Operation:

  1. Check if queue is full (overflow condition)
  2. Increment the rear pointer
  3. Insert the element at queue[rear]

Dequeue Operation:

  1. Check if queue is empty (underflow condition)
  2. Return the value at queue[front]
  3. Increment the front pointer

Time Complexity

Access: O(n)
Search: O(n)
Insertion (Enqueue): O(1)
Deletion (Dequeue): O(1)

Common Operations

Enqueue (add element to rear)
Dequeue (remove element from front)
Front (get front element)
Check if queue is empty or full

Code (C Example)

#include <stdio.h>
#define MAX 5

int queue[MAX], front = -1, rear = -1;

void enqueue(int value) {
    if (rear == MAX-1)
        printf("Queue Overflow\n");
    else {
        if (front == -1) front = 0;
        queue[++rear] = value;
    }
}

void dequeue() {
    if (front == -1 || front > rear)
        printf("Queue Underflow\n");
    else
        printf("Dequeued: %d\n", queue[front++]);
}

int main() {
    enqueue(10);
    enqueue(20);
    enqueue(30);
    dequeue();
    return 0;
}

Output

Dequeued: 10
